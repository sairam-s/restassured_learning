Set %JAVA_HOME% in the environment variable (win) or in the path (mac).
Download IDE (eclipse) if its not available on your machine.
Download all the Rest Assured jars https://github.com/rest-assured/rest-assured/wiki/Downloads
* Remember to download xml and json jars.
Create a workspace (not mandatory) and setup a java project in the IDE.
Add the jars to the project's java build path.
* Remember to unzip the jars in the parent and child directories.
Create a new class.
